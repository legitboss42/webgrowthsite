# Hosting Setup Checklist

Use this checklist to set up fast, secure hosting for a small business website.

## Before you buy

- [ ] Choose a provider with data centers close to your customers
- [ ] Confirm free SSL and HTTP/2 (or HTTP/3)
- [ ] Make sure server-side caching is available
- [ ] Ensure daily backups are included

## After purchase

- [ ] Point your domain DNS to the host
- [ ] Enable SSL (HTTPS) and force redirects
- [ ] Turn on a CDN if available
- [ ] Set up uptime monitoring
- [ ] Run a Lighthouse / Core Web Vitals test

## Security essentials

- [ ] Enable 2FA on hosting + domain accounts
- [ ] Use strong passwords (password manager)
- [ ] Keep your CMS/plugins updated
- [ ] Enable a WAF if your host/CDN provides one
